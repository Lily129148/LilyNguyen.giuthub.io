Python

Python is an essential tool for data analysts due to its versatility and powerful libraries. In my GDP Prediction project, I used pandas library not only to remove all the missing data, but to manipulate it so that the dataset is better to look at. I also use seaborn and matplotlib to visualise the relationship of each variable. And lastly, I built an OLS model to understand more whether the GDP has been effected by any macroeconomics factors and how stong is the affection
Another project that I used Python was Movie Industry. This is a personal projetc. The dataset was collected from Kaggle. In this project, I also use pandas and numpy to clean the data. To understand further the effection of variables on each movie profit, I drew a correlation matrix and use scatter plot to make it easier to visualise.
